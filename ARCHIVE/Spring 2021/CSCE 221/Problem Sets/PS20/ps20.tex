$$Z = (A_1 \times 10^2 + A_2)(A_1 \times 10^2 + A_2)$$
$$Z = ((A_3 + A_4) \times 10^2 + A_2)((A_3 + A_4) \times 10^2 + A_2)$$
which requires 9 miltiplications and k recursive breakdowns. 
$$4^k = 9$$
solving for k
$$k log_2(4) = log_2(9)$$
$$2k = log_2(9)$$
$$k = log_2(9^{\frac{1}{2}})$$
so
$$\boxed{O(n^{log_2(3)})}$$
